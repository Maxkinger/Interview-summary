## 算法总结

### 一、排序

`swap(int[] nums, int i, int j)` 定义如下：

```java
public void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
```

#### 冒泡排序

```java
public void bubbleSort(int[] nums) {
        int len = nums.length;
        for (int i = 0; i < len; i++) {
            for (int j = 0; j < len - i - 1; j++) {
                if (nums[j] > nums[j + 1]) {
                    swap(nums, j, j + 1);
                }
            }
        }
    }
```

#### 选择排序

```java
public void selectSort(int[] nums) {
        int len = nums.length;
        int max = 0;
        for (int i = len - 1; i > 0; i--) {
            for (int j = 0; j <= i; j++) {
                if (nums[j] > nums[max]) {
                    max = j;
                }
            }
            swap(nums, i, max);
        }
    }
```

#### 插入排序

```java
public void insertSort(int[] nums) {
        int len = nums.length;
        for (int i = 1; i < len; i++) {
            for (int j = 0; j < i; j++) {
                if (nums[j] > nums[i]) {
                    int temp = nums[i];
                    System.arraycopy(nums, j, nums, j + 1, i - j);
                    nums[j] = temp;
                }
            }
        }
    }
```

#### 快速排序

```java
public void quickSort(int[] nums, int start, int end) {
        if (start >= end) return;
        int left = start;
        int right = end;
        int flag = nums[left];
        while (left < right) {
            while (nums[right] >= flag && right > left) {
                right--;
            }
            if (left < right) {
                nums[left] = nums[right];
            }
            while (nums[left] <= flag && left < right) {
                left++;
            }
            if (left < right) {
                nums[right] = nums[left];
            }
        }
        nums[left] = flag;
        quickSort(nums, start, left - 1);
        quickSort(nums, left + 1, end);
    }
```

#### 归并排序

```java
public int[] mergeSort(int[] nums) {
        if (nums.length <= 1) return nums;
        int len = nums.length;
        int[] left = new int[len / 2];
        int[] right = new int[len - len / 2];
        System.arraycopy(nums, 0, left, 0, left.length);
        System.arraycopy(nums, left.length, right, 0, right.length);
        return merge(mergeSort(left), mergeSort(right));
    }

    public int[] merge(int[] left, int[] right) {
        int[] res = new int[left.length + right.length];
        int i = 0, j = 0, p = 0;
        while (i < left.length && j < right.length) {
            if (left[i] < right[j]) {
                res[p] = left[i];
                i++;
            } else {
                res[p] = right[j];
                j++;
            }
            p++;
        }
        if (i < left.length) {
            System.arraycopy(left, i, res, p, left.length - i);
        }
        if (j < right.length) {
            System.arraycopy(right, j, res, p, right.length - j);
        }
        return res;
    }

```

#### 堆排序

```java
public void heapSort(int[] nums) {
        int len = nums.length;
        for (int i = len / 2; i >= 0; i--) {
            heapify(nums, i, len);
        }
        for (int i = 0; i < len; i++) {
            swap(nums, 0, len - 1 - i);
            heapify(nums, 0, len - 1 - i);
        }
    }

    public void heapify(int[] nums, int start, int len) {
        if (start >= len) return;
        int left = start * 2 + 1;
        int right = start * 2 + 2;
        int max = start;
        if (left < len && nums[left] > nums[max]) {
            max = left;
        }
        if (right < len && nums[right] > nums[max]) {
            max = right;
        }
        if (max != start) {
            swap(nums, start, max);
            heapify(nums, max, len);
        }
    }
```

为什么堆排序中堆化 heapify 既要上升又要递归地下沉？

假设没有下沉，第一次建堆完成的时候确实也能把整个堆的最大值上浮到顶，但是由于后续只交换而不下沉，造成后续堆底部的大值无法上浮，使排序结果错误。
